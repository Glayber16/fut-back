Back-end de um projeto pessoal sobre ida ao estadio + estudo sobre Nest.js

Conexão com a API [API-Football](https://www.api-football.com/) para pegar os escudos dos times automaticamente
Criação, listagem e remoção de jogos assistidos.
Gerenciamento de amigos que acompanharam nas partidas.
Total de jogos, vitórias, empates e derrotas.
  - Quantas partidas eu fui com algum amigo.
  - Contagem de partidas onde o usuário foi sozinho.
